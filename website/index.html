Motor Vehicle Collision Dashboard

Big Data Analytics menggunakan Hadoop MapReduce dan Random Forest

Project Overview

Projek Big Data Analytics (BDA) ini bertujuan untuk menganalisis dan memprediksi risiko kecelakaan kendaraan bermotor di New York City (NYC) menggunakan dataset historis berskala besar dari NYC Open Data yang mencakup jutaan catatan kecelakaan. Sistem ini mengintegrasikan ekosistem Big Data Hadoop MapReduce untuk pemrosesan paralel terdistribusi, dikombinasikan dengan algoritma Machine Learning Random Forest untuk klasifikasi dan peramalan prediktif.

Metodologi projek ini terbagi menjadi empat tahap utama:

  • Preprocessing Data: Pembersihan dan penyaringan data mentah berukuran gigabyte untuk menangani nilai kosong, duplikasi koordinat, serta penyesuaian format tanggal/waktu kecelakaan.
  • Hadoop MapReduce (Borough Analysis): Pemrosesan paralel terdistribusi menggunakan arsitektur Hadoop untuk menghitung total kecelakaan di setiap wilayah (Borough) dengan efisiensi tinggi.
  • Klasifikasi Risiko (Random Forest Classifier): Pemodelan Machine Learning untuk mengklasifikasikan tingkat keparahan risiko kecelakaan ("High", "Medium", "Low") berdasarkan fitur seperti lokasi, tanggal, faktor penyebab kecelakaan, dan jenis kendaraan yang terlibat, mencapai tingkat akurasi sebesar 62% setelah optimasi fitur.
  • Peramalan Deret Waktu (Time-Series Forecasting): Prediksi tren jumlah kecelakaan harian dan tingkat risiko harian untuk 30 hari ke depan menggunakan pendekatan lag-based regresi autoregresif terawasi, murni memanfaatkan pola data historis yang sudah terjadi sebelumnya.

Melalui visualisasi dashboard interaktif ini, pemangku kepentingan seperti kepolisian (NYPD) dan dinas perhubungan dapat memantau wilayah rawan kecelakaan (hotspot), memahami faktor utama penyebab insiden, serta mengantisipasi tren kecelakaan di masa mendatang guna merancang strategi pencegahan yang lebih efektif.
